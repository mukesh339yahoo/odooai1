Key Features

  • Conditional (Domain) Security: Hide or make fields read-only dynamically based on the record's current state.
  • Button & Action Hiding: Completely hide functional action buttons from specific user groups to prevent unauthorized operations.
  • Export Protection: Stop users from exporting sensitive fields (like phone numbers or margins) to Excel or CSV.
  • View-Type Granularity: Apply rules specifically to Form views, List views, or everywhere globally.
  • Exclude Behavior: Save time by applying rules to everyone except selected administrators or managers.

How to Install & Enable

1. Installation

  • Download the app and extract the advanced_field_security folder into your Odoo addons path.
  • Restart your Odoo server instance.
  • Go to the Apps menu, click Update Apps List, and install the "Advanced Field-Level Security" module.

2. Enabling the App

  • Ensure you have activated Developer Mode in the Odoo settings.
  • Navigate to Settings > Technical > Security > Field Security Rules.
  • Click New to start creating your dynamic UI rules immediately!
